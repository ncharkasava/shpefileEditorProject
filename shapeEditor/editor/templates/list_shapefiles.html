Welcome to ShapeEditor Project

Free Open Source Web Application for Editing Shapefiles
Save your money. Save your time.

Get Started

Upload

Use import shapefile tool to upload your zipped shapefile. Make sure it's in .zip format and has 4 required files in there (.dbf, .prj, .shp, and .shx)

Edit

Once uploaded, you can access your shapefiles anytime. Select, view and edit feature's geometry and attribute's values. Add new features and delete old ones.

Save

Use export shapefile tool to save edited shapefile on your computer. Exported from ShapeEditor shapefiles may be used in any other desktop application.
